## 2020 版本

### 一. Eureka Server

2020.0.0版本，Eureka Server 启动时，需要手动添加依赖

```xml
<dependency>
  	<groupId>com.sun.jersey.contribs</groupId>
    <artifactId>jersey-apache-client4</artifactId>
    <version>1.19.4</version>
</dependency>
```
